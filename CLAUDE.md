## UI: Lime

This project uses the **Lime** UI kit. A Claude Code skill is bundled at
`.claude/skills/lime` (skill name: `lime-ui`). **Use it whenever you build or
style UI** — it has the kit's components, tokens, and rules so the result stays
consistent and cheap to produce.

- Build with the kit's tokens and components; don't invent new colors or one-off components.
- Human usage guide: `USAGE.md`.
- Validate the manifest after changes: `npx uikit-cli validate`.
